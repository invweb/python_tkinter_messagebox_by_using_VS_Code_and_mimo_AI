# Concat App

Приложение для объединения текста из двух полей ввода.

## Description (English)

A simple desktop application for concatenating text from two input fields. The app features a clean white interface with highlighted input fields. If either field is empty, a warning notification appears. The merged result is displayed below the button.

### Technologies

- **Python 3** — programming language
- **tkinter** — built-in GUI library
- **messagebox** — notification module from tkinter

## Описание (Русский)

Простое десктопное приложение для объединения текста из двух полей ввода. Интерфейс выполнен в белом цвете с подсвеченными полями ввода. Если хотя бы одно поле пустое, появляется предупреждение. Результат объединения отображается ниже кнопки.

### Технологии

- **Python 3** — язык программирования
- **tkinter** — встроенная библиотека для создания графического интерфейса
- **messagebox** — модуль tkinter для вывода уведомлений

## Beschreibung (Deutsch)

Eine einfache Desktop-Anwendung zum Zusammenführen von Text aus zwei Eingabefeldern. Die App bietet eine saubere weiße Oberfläche mit hervorgehobenen Eingabefeldern. Wenn eines der Felder leer ist, wird eine Warnung angezeigt. Das zusammengeführte Ergebnis wird unter dem Button angezeigt.

### Technologien

- **Python 3** — Programmiersprache
- **tkinter** — eingebaute GUI-Bibliothek
- **messagebox** — Benachrichtigungsmodul aus tkinter
